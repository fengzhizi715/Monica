## 🚀 快速开始

```bash
# 进入工具目录
cd i18n

# 一键检查所有问题
./quick_check.sh

# 或者单独使用各个工具
./string_manager.sh -m    # 检查同步性
./string_manager.sh -c -d # 检查重复项
./position_check.sh       # 检查行号一致性
```
